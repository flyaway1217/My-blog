# 千年金融史

## 综述

**金融的四个关键要素:**

1. 在时间上重新配置经济价值
2. 重新配置风险
3. 重新配置资本
4. 扩展了资源重新配置的渠道和复杂程度

金融契约通常在两种人中寻找平衡:

1, 希望将价值折现的人
2. 希望将价值转换到未来的人

## 古典文明

文字的出现来源于金融记录的需求。

最早的金融契约(最早的金融工具): 印玺

最早的复利债务: 两个苏美尔古城之间的战争赔偿

最早利用借贷取得政治利益: 大流士二世，他利用借来的钱打赢了内战，取得王位。

雅典高度依赖粮食进口，而良好的金融制度保障了粮食进口。
