# 千年金融史

## 综述

### 金融的四个关键要素

1. 在时间上重新配置经济价值
2. 重新配置风险
3. 重新配置资本
4. 扩展了资源重新配置的渠道和复杂程度

金融契约通常在两种人中寻找平衡:

1, 希望将价值折现的人
2. 希望将价值转换到未来的人
